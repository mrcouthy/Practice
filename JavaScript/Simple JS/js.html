Hoisting is JavaScript's default behavior of moving all declarations to the top of the current scope (to the top of the current script or the current function).
"use strict"; Defines that JavaScript code should be executed in "strict mode".

Use Parameter Defaults If a function is called with a missing argument, the value of the missing argument is set to undefined. Undefined values can break your code. It is a good habit to assign default values to arguments. Example function myFunction(x, y) { if (y === undefined) { y = 0; } } Or, even simpler: function myFunction(x, y) { y = y || 0;//if y is undifined use 0 }
